# Relevancy V2 Frozen

Relevancy prompt/rubric v2 is frozen for production use.

- `prompt_version`: `v2`
- `rubric_version`: `relevancy_rubric_v2`
- prompt hashes are recorded in run artifacts and DB `prompt_versions_json`.

Policy: Any semantic/rubric change must be introduced as `v3` (do not modify v2 behavior in place).
